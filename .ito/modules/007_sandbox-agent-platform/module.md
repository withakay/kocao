# Sandbox Agent Platform

## Purpose
Adopt `sandbox-agent` as Kocao's single in-sandbox coding-agent control surface so the platform can launch, interact with, and manage OpenCode, Claude, Codex, and Pi sessions through one API while preserving Kubernetes-native orchestration, auth, audit, and UI flows.

## Scope
- sandbox-agent-runtime
- agent-session-ui
- control-plane-api
- harness-runtime
- session-durability
- security-posture
